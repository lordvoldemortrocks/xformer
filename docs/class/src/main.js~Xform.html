    public execute(query: Array | Object, data: any, ext: Object): Object source

    Takes a query and data as input and executes all pipelines within the query, with each pipeline receiving the provided data. A query can be a list of pipes or a JSON structure. A pipe is an array of action descriptions. An action can be described as a string or JSON.

    Read docs for the available action palette.

    Params:

    NameTypeAttributeDescription
    query Array | Object
    data any
    ext Object

    = {}

    Return:

    Object

    Example:

         E.execute({ avg_by_30: ['getAvg', 'getRate(30)'] }, [1, 2 , 3]);

     //=>
     {
       avg_by_30: {
         buffer: [
           { title: 'Original Data', data: [1, 2, 3] },
           { title: 'getAvg', data: 2 },
           { title: 'getRate(30)', data: 0.067 }
         ],
         result: 0.067
       }
     }
