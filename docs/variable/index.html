    public defaultAll(value: number, data: Object | Array): number: * source 

    import {defaultAll} from '@muhammadkasim/xformer/src/palette.js'

    Recieves an array or object and replaces each junky value with the provided fallback value. A value is considered junk if it can not be converted to a valid number. A stringy number is converted to number.

    Return:

    number

    Example:

         defaultAll('N/A', [1, NaN, '3'])) //=> [1, 'N/A', 3]

    public pickFrom(path: Array<string>, input: Object): any: * source 

    import {pickFrom} from '@muhammadkasim/xformer/src/palette.js'

    Retrieves value at the specified path from a JSON object. '*' in the path is regarded as a wildcard, meaning anything at this level will used to pick value from.

    Return:

    any

    Example:

         pickFrom(['a','b','c'], {'a': {'b': {'c': 20}}}) //=> 20
     pickFrom(['a','*','x'], {'a': {'b': {'x': 20}, 'c': {'x': 40}}}) //=> [20, 40]

    public runAll(pipes: Array): Array: * source 

    import {runAll} from '@muhammadkasim/xformer/src/palette.js'

    Execute a list of pipelines on provided data. A pipeline is an array of action descriptions. An action can be described in form of a string or JSON.

    Return:

    Array

    Example:

         runAll([
       ['pickFrom(["alpha"])', 'getAvg'], // pipe 1
       ['pickFrom(["beta"])', 'getAvg'],  // pipe 2
     ], {
       alpha: { a1: 1, a2: 2, a3: 3 },
       beta: { b1: 11, b2: 22, b3: 33 },
     })

     => [2, 22]
